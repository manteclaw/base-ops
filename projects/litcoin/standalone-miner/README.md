# Manteclaw Standalone Litcoiin Miner

Autonomous mining agent for the Litcoiin protocol on Base L2.

## Features
- Self-contained mining loop — no external orchestration needed
- Smart model routing with inclusionai/ling-2.6-1t:free as primary
- Circuit breaker protection for gas optimization
- GitHub auto-commit for solution tracking
- Bankr wallet integration for earnings

## Quick Start
```bash
pip install litcoin bankr-sdk
python miner.py --bankr-key YOUR_BANKR_KEY --ai-key YOUR_OPENROUTER_KEY
```
