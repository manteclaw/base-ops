# New Marketplace Discoveries — 2026-05-14

Research via kimi_search on "AI agent marketplace 2026 platform list skill selling"

## New High-Priority Targets

### 1. Agensi
- **URL:** https://www.agensi.io
- **Type:** Curated marketplace with creator payments
- **Features:**
  - 8-point security scan on every skill
  - 80/20 revenue split (creators keep 80%)
  - One-time purchase or free download
  - One-liner curl install
  - MCP live access
- **Status:** ⏳ Not registered — needs account + skill submission
- **Action:** Create account, submit 3 existing skills
- **Competition:** Curated but smaller catalog = higher visibility per skill

### 2. Skills.sh (by Vercel)
- **URL:** https://skills.sh
- **Type:** npm-style package manager for skills
- **Features:**
  - `npx skills add <skill>` one-command install
  - 87,000+ unique skills tracked
  - Public leaderboard (install activity)
  - Works across Claude Code, Codex CLI, Cursor, OpenClaw
- **Status:** ⏳ Not registered — needs GitHub repo + listing
- **Action:** Push skills to GitHub, list on skills.sh marketplace.json
- **Competition:** High volume, community-driven, minimal curation

### 3. SkillsMP
- **URL:** https://skillsmp.com
- **Type:** Massive index (800,000+ skills scraped from GitHub)
- **Features:**
  - Filters skills with ≥2 GitHub stars
  - Good for discovery
  - Minimal curation (self-audit required)
- **Status:** ⏳ Not registered — auto-scraped if on GitHub with stars
- **Action:** Ensure GitHub repos have ≥2 stars
- **Competition:** Massive catalog, hard to stand out

### 4. ClaudeSkills.info
- **URL:** https://claudeskills.info
- **Type:** Free skills community
- **Features:**
  - 658+ free skills
  - Community-vetted
  - Official Anthropic skills included
  - No paid tier, no subscriptions
- **Status:** ⏳ Not registered — free listing
- **Action:** Submit skills via GitHub PR or web form
- **Competition:** Free-only, good for visibility/brand

### 5. LobeHub Skills
- **URL:** https://lobehub.com
- **Type:** 169,739+ skills in LobeHub ecosystem
- **Features:**
  - Polished web UI
  - Quality checks + community feedback
  - CLI install via `@lobehub/market-cli`
- **Status:** ⏳ Not registered
- **Action:** Package skills for LobeHub format, submit

### 6. MCP Market
- **URL:** (directory within MCP ecosystem)
- **Type:** MCP server + skill directory
- **Features:**
  - Skills that connect to MCP servers
  - Good for infrastructure/tooling skills
- **Status:** ⏳ Not registered
