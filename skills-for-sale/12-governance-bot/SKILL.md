# Skill: Governance Vote Bot

## Overview
Automated Snapshot governance voting for DAO delegates. Polls proposals, signs EIP-712 votes, executes based on configurable rules.

## What You Get
- `governance_bot.py` + `snapshot_client.py` + `voting_engine.py`
- Snapshot GraphQL polling
- EIP-712 vote signing
- Configurable rule engine (vote with delegator, abstain on controversial, etc.)
- Structured logging + audit trail

## Installation
```bash
git clone https://github.com/manteclaw/base-ops
cd base-ops/projects/governance-bot
python3 governance_bot.py --config config.yaml
```
